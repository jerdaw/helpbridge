## Implementation Notes

### Messaging Approaches

Templates support multiple messaging strategies:

- **Story-Driven**: Empathy-first narrative ("Imagine this: You need to find a food bank...")
- **Data-Driven**: Metrics and features ("196+ verified services, 7 languages")
- **Crisis-Focused**: Immediate help framing ("If you're searching 'I need food today'...")
- **Professional**: For social workers and case managers (LinkedIn-appropriate)

### Design Philosophy

All specifications prioritize:

- **Accessibility**: WCAG AA compliance throughout
- **Community-Centered**: Inclusive, warm, approachable (not corporate)
- **Privacy-First**: Messaging emphasizes no-tracking, on-device search
- **Multilingual-Friendly**: Visual assets avoid text-heavy designs

### Budget Flexibility

Assets guide provides 3 budget tiers:

1. **DIY** (Free): 10-20 hours using Figma/Canva
2. **Freelancer** ($500-2,000): 2-3 weeks professional work
3. **Agency** ($5,000-20,000): 4-8 weeks comprehensive brand identity

Recommendation: Start DIY or freelancer for beta, invest in agency if project scales.
